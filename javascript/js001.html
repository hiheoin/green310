자바스크립트_연산자

설명문은 document(보이는 화면 전체)에서 확인할 수 있고, 자바스크립트 예제는 console에서 확인할 수 있다.

산술 연산자

  • 더하기 : +
  • 빼기 : -
  • 곱하기 : *
  • 나누기 : /
  • 나머지 : %

<작성법>
변수로 숫자를 지정한다.
let 또는 var 또는 const로 변수를 말한다.

let - 가장 유연하게 사용 가능, 모든 변수 이름 앞에 사용 가능, 에러가 가장 없음
var - 처음으로 생긴 변수 지정법으로 가장 오랜 역사를 가짐, 많은 오픈파일에서 보일 수 있음 but 중복되더라도 발견되지 못 한다는 단점
const - 고유명사 같은 개념으로 변수를 바꿀 수 없음, 대부분 선택자의 이름으로 한 번 할당되면 바뀌지 않는 값에 씀
let a = 10
(풀이 : a는 10이다.)
let b = 5
(풀이 : b는 5이다.)
let c ;
(풀이 : c를 변수 이름으로 사용하겠다.)
c=a+b
(풀이 : c는 10+5이다.)

비교 연산자

변수가 두가지 필요하고, 2개의 데이터를 부등호를 통해 비교하며, 바르면 true 틀리면 false가 출력된다.

  • A가 B보다 크다 : A>B
  • A가 B보다 크거나 같다 : A>=B
  • A가 B와 같다 : A==B
  • A가 B와 다르다 : A!=B
  • A가 50보다 크거나 20보다 작다
    => A>50 && A<20

논리 연산자

  • 그리고(and) : &&
  • 또한(or) : \\
  • 아니다(not) : !

예제풀이

나이를 물은 후 20세 이상이면 true, 20세 이하면 false가 나타나게 해라.
만약에 age가 20보다 크면 "당신은 성인입니다."라는 말을 document에 출력하고,
그렇지 않으면 "당신은 성인이 아닙니다."라는 말을 출력해라.

증감 연산자

변수에 저장된 값에 1씩 증가하거나 감소한다.

  • 증가 : ++
  • 감소 : --

대입 연산자

연산된 데이터를 지정된 변수에 대입한다.

  • = 은 뒤에 있는 숫자나 내용을 앞의 변수에 대입한다.
  • 변수 =5
    (풀이 : 5는 변수다.)
    변수 +=5
    (풀이 : 5를 더해서 변수에 대입해라.)
