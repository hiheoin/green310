박스속성

웹페이지 레이아웃시 가장 중요한 속성

  • 요소 크기 설정 - width, height
  • 영역 설정 - padding
  • 외곽선 설정 - border
  • 여백 설정 - margin

작성법

  • 전체적용
    - 숫자와 단위를 함께 하나만 작성

  • 각자적용
    - 숫자와 단위를 4번 써서 작성, 순서는 위/오른쪽/아래/왼쪽
    - 작성시 쉼표를 쓰지 않고 띄어쓰기를 통해 작성

  • 세트적용
    - 위아래/왼쪽오른쪽, 숫자와 단위를 두 번 써서 작성
    - 앞이 위아래/뒤가 왼쪽오른쪽, 각자적용과 마찬가지로 띄어쓰기를 통해 작성

  • 부분적용
    • margin-top : 위쪽 여백
    • margin-botttom : 아래쪽 여백
    • margin-left : 왼쪽 여백
    • margin-right : 오른쪽 여백
    • padding-top : 위쪽 영역
    • padding-bottom : 아래쪽 영역
    • padding-left : 왼쪽 영역
    • padding-right : 오른쪽 영역

border 설정방법

- 박스의 테두리를 말하며, padding 바깥쪽에 위치
- 색상, 굵기, 모양으로 설정

  • border-color : 색상기재
  • border-width : 숫자와 단위로 표시
  • border-style
    • none - 선을 없앰
    • solid - 실선
    • dotted - 동그라미점으로 점선만들기(간격설정 안 됨)
    • dashed - 긴사각형 점선
    • double - 실선 두 줄
    • groove - (액자형) 테두리가 파인 것처럼 그려짐
    • ridge - (액자형) 테두리가 튀어나온 것처럼 그려짐 / groove 반대
    • inset - (액자형) 내부 요소가 파인 것처럼 그려짐
    • outset - (액자형) 내부 요소가 튀어나온 것처럼 그려짐 / inset 반대

작성법
- color, width, style을 한 번에 쓸 수도 있고 각자 쓸 수도 있음
- 순번에 관계없음
ex) border : red 2px solid (2px 너비의 빨간색 실선)
