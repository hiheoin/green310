박스속성

웹페이지 레이아웃시 가장 중요한 속성

  • 요소 크기 설정 - width, height
  • 영역 설정 - padding
  • 외곽선 설정 - border
  • 여백 설정 - margin

작성법

  • 전체적용
    - 숫자와 단위를 함께 하나만 작성

  • 각자적용
    - 숫자와 단위를 4번 써서 작성, 순서는 위/오른쪽/아래/왼쪽
    - 작성시 쉼표를 쓰지 않고 띄어쓰기를 통해 작성

  • 세트적용
    - 위아래/왼쪽오른쪽, 숫자와 단위를 두 번 써서 작성
    - 앞이 위아래/뒤가 왼쪽오른쪽, 각자적용과 마찬가지로 띄어쓰기를 통해 작성

  • 부분적용
    • margin-top : 위쪽 여백
    • margin-botttom : 아래쪽 여백
    • margin-left : 왼쪽 여백
    • margin-right : 오른쪽 여백
    • padding-top : 위쪽 영역
    • padding-bottom : 아래쪽 영역
    • padding-left : 왼쪽 영역
    • padding-right : 오른쪽 영역

border 설정방법

- 박스의 테두리를 말하며, padding 바깥쪽에 위치
- 색상, 굵기, 모양으로 설정

  • border-color : 색상기재
  • border-width : 숫자와 단위로 표시
  • border-style
    • none - 선을 없앰
    • solid - 실선
    • dotted - 동그라미점으로 점선만들기(간격설정 안 됨)
    • dashed - 긴사각형 점선
    • double - 실선 두 줄
    • groove - (액자형) 테두리가 파인 것처럼 그려짐
    • ridge - (액자형) 테두리가 튀어나온 것처럼 그려짐 / groove 반대
    • inset - (액자형) 내부 요소가 파인 것처럼 그려짐
    • outset - (액자형) 내부 요소가 튀어나온 것처럼 그려짐 / inset 반대

작성법
- color, width, style을 한 번에 쓸 수도 있고 각자 쓸 수도 있음
- 순번에 관계없음
ex) border : red 2px solid (2px 너비의 빨간색 실선)

백그라운드 (배경설정)

- 배경색이나 배경이미지를 삽입할 때 사용
- 많은 속성들을 한 번에 기재할 수도 있고, 별도로 기재할 수도 있음

  • background-color
    • 영문 이름으로 작성 : red, black 등
    • hex코드로 쓰는 방법 : #ffffff, #000000, #fff 등
      반복으로 들어갈 경우 레드에서 하나, 그린에서 하나, 블루에서 하나씩 작성 가능(#fff)
    • RGB 수치값 작성 : rgb(0,50,20)
    • RGB와 투명도 같이 작성 : rgba(0, 50, 200, 0.5)
  • background-image : url(이미지경로 작성)
  • background-repeat : 배경이미지 또는 컬러를 반복
    • repeat - 가로, 세로형으로 모두 패턴처럼 깔림(바닥)
    • repeat-x - 가로 방향으로만 깔림(벽지)
    • repeat-y - 세로 방향으로만 깔림
    • no-repeat - 반복없음
  • background-size : 배경이미지의 크기를 태그 사이즈 대비 변경하고자 할 때 사용
    • contain - 이미지의 비율을 유지하면서 영역을 꽉 채움
    • cover - 배경이미지가 잘리더라도 화면의 영역을 꽉 채움
  • background-position : top, center, left 등 위치를 지칭하는 단어로도 사용 가능
    숫자와 단위로도 이동 가능
    왼쪽 상단에서부터 거리가 멀어지고, 앞 수가 오른쪽으로 이동, 다음 수가 아래쪽으로 이동(20px 60px)
  • background-attachment : fixed로 바닥에 붙이는 효과

그라디언트를 활용하여 배경색 또는 패턴을 제작할 수 있다

  • 직선형 그라디언트 : linear-gradient(각도,시작색깔,마지막색깔)
  • 방사형 그라디언트 : radial-gradient(시작색깔,마지막색깔)
  • patten - 직선 그라데이션의 수직시작점을 %값으로 잡아 옆에 여백을 만들어 반복시켜 적용
